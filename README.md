# CompanyIntel

Free US company due diligence & intelligence platform. Search any company and get a comprehensive report aggregating data from government sources.

## Data Sources

- **SEC EDGAR** - Company filings, financial data, officers
- **CFPB** - Consumer complaint history and trends
- **EPA ECHO** - Environmental compliance and violations
- **OSHA** - Workplace safety inspections and penalties
- **USPTO** - Patent portfolio (requires API key for full access)
- **FDIC BankFind** - Banking health data for financial institutions

## Tech Stack

- Next.js 16 with App Router
- TypeScript
- Tailwind CSS
- Deployed on Fly.io

## Getting Started

```bash
npm install
npm run dev
```

## Deployment

```bash
fly deploy
```
